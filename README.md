# Report-on-Biological-computation This repository holds a data science pipeline designed to study the effect of mutations in the promoter region on gene expression in 24 different datasets. The current research combines mRNA and protein expression levels of these three types of mutations: substitutions, insertions, and deletions. To perform such an analysis, libraries such as pandas, glob, and Seaborn from the Python programming language were used. The file merging was conducted using an automated pipeline where gene names were recognized with the help of regular expressions, resulting in one master file combining all the datasets. Visualization techniques, which included boxplot, scatter, and histogram plots with KDE overlay, helped to pick five genes to be validated through experiments.
